cap2-4問2 contents テーブルから 前 現在 次のコンテンツ名を取り出し昇順に表示せよ

MariaDB [workbook]> select
    ->
    -> past.c_title AS 前コンテンツ名,
    ->
    -> now.c_title AS 現コンテンツ名,
    ->
    -> futu.c_title AS 次コンテンツ名
    ->
    -> from(
    ->
    -> contents AS past
    ->
    -> INNER JOIN
    ->
    -> contents AS now
    ->
    -> ON
    ->
    -> past.next_id = now.c_id
    -> )
    ->
    -> INNER JOIN
    ->
    -> contents AS futu
    ->
    -> ON
    ->
    -> now.next_id = futu.c_id
    ->
    -> order by past.c_id ASC
    -> ;
